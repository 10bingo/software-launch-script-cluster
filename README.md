# software-launch-script-cluster
集群中常用软件的启动脚本，避免重复启动的繁琐



## vimic配置文件

- 显示中文帮助

- 修改了字体

- 新建 .sh 文件，自动插入文件头信息

- 新建文件后，自动定位到文件末尾

- 自动保存

- 语法高亮

- 显示行号 

- 突出显示当前行
